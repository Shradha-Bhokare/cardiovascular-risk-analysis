# Cardiovascular Disease Risk Analysis

## Project Overview
This project analyzes cardiovascular disease risk factors using SQL and Power BI to identify patterns related to lifestyle and health conditions.

## Tools Used
- SQL
- Power BI

## Key Analysis
- Analyzed the impact of smoking and alcohol consumption
- Evaluated BMI and physical activity patterns
- Compared cardiovascular risk across age groups
- Identified major lifestyle-related risk factors

## Dashboard Features
- KPI cards
- Risk factor analysis
- Lifestyle comparison charts
- Age group analysis
- Interactive dashboard filters

## Files Included
- Dataset
- SQL Queries
- Power BI Dashboard
